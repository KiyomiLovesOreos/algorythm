📋 OVERVIEW
───────────────────────────────────────────────────────────────────
This update adds extensive new synthesis methods, audio effects,
and sound design capabilities to Algorythm.

🎹 NEW INSTRUMENTS (3)
───────────────────────────────────────────────────────────────────
1. PhysicalModelSynth
   • String modeling (guitar, harp)
   • Drum modeling (percussion)
   • Wind modeling (flute, clarinet)
   • Uses Karplus-Strong algorithm and physical modeling

2. AdditiveeSynth
   • Combines multiple sine wave harmonics
   • Perfect for organ sounds and rich timbres
   • Configurable harmonic amplitudes

3. PadSynth
   • Multiple detuned oscillator voices
   • Creates lush, thick pad sounds
   • Configurable voice count and detune amount

🎚️ NEW EFFECTS (15+)
───────────────────────────────────────────────────────────────────
Time-Based:
  • Reverb - Space and depth
  • Delay - Echo with feedback

Modulation:
  • Chorus - Thickening effect
  • Flanger - Jet plane swoosh
  • Phaser - Notch sweeps
  • Tremolo - Amplitude modulation
  • Vibrato - Pitch modulation
  • AutoPan - Stereo movement

Distortion:
  • Distortion - Waveshaping with tone control
  • Overdrive - Smooth tube saturation
  • Fuzz - Extreme clipping

Dynamics:
  • Compressor - Dynamic range control
  • Limiter - Peak limiting
  • Gate - Noise gate

Special:
  • RingModulator - Metallic effects
  • BitCrusher - Lo-fi digital distortion
  • EffectChain - Combine multiple effects

💡 QUICK START
───────────────────────────────────────────────────────────────────
from algorythm import *
from algorythm.effects import *

# New instruments
guitar = SynthPresets.guitar()
strings = SynthPresets.strings()

# New effects
reverb = ReverbFX(room_size=0.7, wet_level=0.3)
delay = DelayFX(delay_time=0.5, feedback=0.5)

# Effect chain
chain = FXChain()
chain.add_effect(DistortionFX(drive=3.0))
chain.add_effect(ReverbFX(room_size=0.8))

# Generate and process
audio = guitar.generate_note(440.0, 2.0)
processed = chain.apply(audio)
