PROBLEM REPORTED:
════════════════

"frequency bar and spectrogram audio don't actually visualize anything"

ROOT CAUSE:
══════════

FrequencyScopeVisualizer and SpectrogramVisualizer were missing the
to_image_data() method required to convert their data into renderable
images. They could generate the raw spectrum/spectrogram data but 
couldn't turn it into visual output.

SOLUTION IMPLEMENTED:
════════════════════

1. FrequencyScopeVisualizer.to_image_data()
   • Takes audio signal chunk
   • Generates FFT spectrum
   • Filters to frequency range (20Hz-20kHz default)
   • Normalizes magnitude to 0-1
   • Draws vertical frequency bars
   • Returns 2D image array

2. SpectrogramVisualizer.to_image_data()
   • Takes audio signal chunk
   • Generates STFT spectrogram
   • Resizes to target dimensions
   • Normalizes magnitude to 0-1
   • Flips vertically (low freq at bottom)
   • Returns 2D image array

TECHNICAL DETAILS:
═════════════════

FrequencyScopeVisualizer:
  • Uses np.fft.rfft() for frequency analysis
  • Hanning window for spectral leakage reduction
  • Converts to dB scale (20 * log10)
  • Interpolates to image width
  • Draws bars from bottom up

SpectrogramVisualizer:
  • Uses Short-Time Fourier Transform (STFT)
  • Configurable window size (default 2048)
  • Configurable hop size (default 512)
  • Converts to dB scale
  • Resizes with nearest-neighbor interpolation
  • Flips vertically for proper orientation

USAGE EXAMPLES:
══════════════

Frequency Scope:
```python
from algorythm.visualization import FrequencyScopeVisualizer

viz = FrequencyScopeVisualizer(
    sample_rate=44100,
    fft_size=2048,
    freq_range=(50, 2000)
)
image = viz.to_image_data(signal, height=720, width=1280)
```

Spectrogram:
```python
from algorythm.visualization import SpectrogramVisualizer

viz = SpectrogramVisualizer(
    sample_rate=44100,
    window_size=2048,
    hop_size=512
)
image = viz.to_image_data(signal, height=720, width=1280)
```

Export as Video:
```python
from algorythm.export import Exporter

exporter = Exporter()
exporter.export(
    signal,
    'output.mp4',
    visualizer=viz,
    video_width=1920,
    video_height=1080
)
```
